Projet Traitement de données

Lors de ce premier semestre dans le BUT Réseaux et Télécoms, nous avons réalisé un premier projet qui alliait Programmation, Système et Réseaux.

Le but de ce projet était de collecter des données sur une machine, de les extraire vers un fichier CSV et enfin de les récupérer avec Python pour les exploiter.

Pour ma part, je devais récupérer la taille que chaque utilisateur occupait et je devais faire varier cette taille afin des créer des graphiques qui représenteraient la taille occupée par chaque utilisateur en fonction du temps.

Nous avons pu réaliser ce projet grâce aux connaissances acquises lors des cours de programmation, de réseaux et de systèmes auxquels nous avons assisté depuis le début de l’année.

Je vais maintenant vous expliquer comment j’ai réalisé ce projet.

Dans un premier temps, j’ai dû installer le module « quota » sur ma machine virtuelle. Ce module permet de gérer les quotas d’utilisation de chaque utilisateur présent sur le système. J’ai ensuite créé des utilisateurs afin d’obtenir davantage de données. Puis j’ai créé un script bash qui permettait de récolter les informations de chaque utilisateur ainsi que leurs noms et le temps auquel cette capture a été faite. Toutes ces informations sont ensuite envoyées vers un fichier CSV. Il fallait ensuite faire varier ces données, j’ai donc écrit un nouveau script bash qui permettait de créer un fichier d’une taille choisie sur le disque de l’utilisateur choisie. Avec le module « crontab » on avait la possibilité de faire répéter les scripts tous les x temps. Cela m’a permis de générer un grand nombre de données sans avoir à le faire manuellement et ce pour les différents utilisateurs.

Après avoir récoltés suffisamment de données, j’ai commencé à écrire mon programme python.

Dans ce dernier, j’ai importé un module permettant de faire des graphiques et de nombreuses autres choses liées aux mathématiques : « matplotlib ». J’ai ensuite affecté chaque donnée de mon fichier dans un dictionnaires afin d’affecter a chaque utilisateurs toutes les données qui lui correspond et enfin j’ai pu réaliser les graphiques. J’ai en fait un en camembert afin de voir quel espace chaque utilisateur occupait par rapport aux autres. Pour finir, j’ai fait un graphique qui retraçait l’évolution de l’espace occupé par chaque utilisateur en fonction du temps.
